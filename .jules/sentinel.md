## 2026-05-09 - [XSS] Fixed LocalStorage XSS via inline event handlers\n**Vulnerability:** XSS vulnerability in DOM rendering via innerHTML and inline event handlers where attributes were insufficiently escaped.\n**Learning:** Using escapeHTML inside inline event handlers (e.g. onclick) is unsafe because the HTML parser decodes entities back to characters before JavaScript execution, which could allow quote breakout.\n**Prevention:** Completely remove inline event handlers. Render variables safely into `data-*` attributes using escapeHTML, and then attach external event listeners to the DOM elements.
